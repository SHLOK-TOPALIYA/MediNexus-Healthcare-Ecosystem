# 🏥 MediNexus Healthcare Ecosystem

A comprehensive healthcare database management system designed using **PostgreSQL**, featuring Entity-Relationship (ER) modeling, relational schema design, BCNF normalization, and SQL implementation.

---

## 📌 Project Overview

MediNexus is a healthcare database system developed as part of the **Database Management Systems (DBMS)** course. The project focuses on designing a scalable and normalized database to efficiently manage healthcare-related information while ensuring data integrity and minimizing redundancy.

---

## 🎯 Objectives

- Design a healthcare database using ER modeling.
- Convert the ER model into a relational schema.
- Normalize the database up to BCNF.
- Implement the schema using SQL.
- Maintain data integrity through constraints and relationships.
